Download EduBots

Quick Start: The easiest way to get started is with Docker. Install Docker, then run the command below to start EduBots in minutes!

Docker Installation (Recommended)

Prerequisites

  • Docker Desktop installed (Download Docker)
  • At least 4GB RAM available
  • Internet connection for first-time setup

Quick Start Command

# Pull and run EduBots
docker run -d --name edubot -p 1999:1999 openroberta/lab

# Access the platform at:
# http://localhost:1999
‚úÖ That's it! Open your browser and navigate to http://localhost:1999

Useful Docker Commands

# Stop the container
docker stop edubot

# Start the container
docker start edubot

# View logs
docker logs edubot

# Remove the container
docker rm edubot

# Pull latest version
docker pull openroberta/lab

  4. Add user to dialout group: sudo usermod -a -G dialout $USER

Cloud Deployment

You can deploy EduBots to cloud platforms for team or classroom access:

Google Cloud Run

# Tag and push to Google Container Registry
docker tag openroberta/lab gcr.io/YOUR-PROJECT/edubot:latest
docker push gcr.io/YOUR-PROJECT/edubot:latest

# Deploy to Cloud Run
gcloud run deploy edubot \
  --image gcr.io/YOUR-PROJECT/edubot:latest \
  --platform managed \
  --region us-central1 \
  --allow-unauthenticated \
  --port 1999 \
  --memory 2Gi

Other Platforms

  • Heroku: Use the Docker deployment method
  • AWS ECS: Deploy as a container service
  • Azure Container Instances: Deploy from Docker Hub
  • DigitalOcean App Platform: Deploy from GitHub

System Requirements

Server Requirements

  • CPU: 2 cores minimum, 4 cores recommended
  • RAM: 2GB minimum, 4GB recommended
  • Storage: 2GB free space
  • OS: Linux, macOS, or Windows with Docker

Client Requirements (Browser)

  • Browser: Chrome, Firefox, Safari, or Edge (latest versions)
  • JavaScript: Must be enabled
  • Screen Resolution: 1024x768 minimum, 1920x1080 recommended
